## Screenshot Guidelines

### Capture Settings
- **Browser**: Chrome (latest version)
- **Resolution**: Minimum 1920x1080
- **Format**: PNG (for transparency support)
- **Zoom**: 100% (default browser zoom)

### Content Guidelines
- Use realistic but non-sensitive sample data
- Show clean, professional UI
- Capture full browser window (not just component)
- Include Salesforce header for context
- Ensure all text is readable

### Sample Data to Use
- Project Name: "Customer Portal Q3 Implementation"
- User: "Demo User"
- Dates: Use current quarter dates
- Avoid real customer names or sensitive data

### Tool Recommendations
- **macOS**: Shift + Cmd + 4 (screenshot selection)
- **Windows**: Snipping Tool or Win + Shift + S
- **Chrome Extension**: Full Page Screen Capture (for scrolling screenshots)

## Editing Screenshots

### Optional Enhancements
1. Add subtle drop shadow
2. Crop to focus area (if needed)
3. Annotate with arrows/callouts (for tutorial docs)
4. Compress to reduce file size (<500KB per image)

### Tools
- **macOS**: Preview (built-in)
- **Windows**: Paint or Paint 3D (built-in)
- **Cross-platform**: GIMP, Photoshop, Figma
